# Server

This is the server component of the Mangaland Fullstack project.

## Installation

1. To install the necessary dependencies, run the following command:

   ```
   npm install
   npm run dev
   ```

2. Now your backend on the 8080 port

   ```
   open http://localhost:8080
   ```
